# Release notes `v0.1.0-alpha.2`

## Stato della release

Questa è una pre-release `alpha` di `metanalisys`.

## Ambito della release

La release `v0.1.0-alpha.2` documenta esclusivamente miglioramenti grafici della GUI successivi alla `v0.1.0-alpha.1`.

Questa release:

- non modifica il core di analisi;
- non modifica la validazione scientifica alpha;
- non modifica il supporto ai formati Office.

## Miglioramenti della GUI

La release aggiunge o migliora:

- icona applicazione nella GUI;
- finestra `Informazioni Software` aggiornata con nome `metanalisys`, stato alpha e licenza GPLv3;
- watermark grafico discreto nell'area report quando il report è vuoto.

## Continuità funzionale

Restano invariati:

- il motore di analisi condiviso;
- la copertura funzionale dei formati Office già supportati;
- la validazione scientifica alpha già documentata;
- la suite di test automatica e la CI del progetto.

## Avvertenza

Il software resta in stato `alpha` ed è destinato a supporto tecnico, sperimentazione controllata e revisione del progetto.

Non sostituisce:

- autorizzazioni;
- catena di custodia;
- verbali;
- procedure operative;
- valutazione professionale del caso concreto.
